# 发音转换器改进版
## 发音转换器是什么
发音转换器，输入中文发音返回同声调对应的外语，以达到用外语说中文的效果。
  
  
我的使用地址：https://fyzhq.dfjgmc.com/
  
改进：
  
1、加载速度优化
  
2、实时转换，输入中文，直接显示转换好的内容
  
3、增加语音预览，可以收听转换好的内容
  
4、点击转换的内容自动选中，方便复制
  
由于原项目使用太痛苦，我才做的改进，现在使用起来才算正常人水准。
